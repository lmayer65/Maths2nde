⚡ Fiche 1 — Partie 2 : Puissances (révisions)

📘 Rappels

a4 = a × a × a × a

De façon générale :

an = a × a × a × a × … × a (n fois)

a est un nombre non nul et n est un entier non nul.

  • a1 = a
  • a0 = 1
  • 0n = 0
  • 1n = 1

⚠️ Attention aux signes !

Ne pas confondre :

(−3)4 = (−3) × (−3) × (−3) × (−3) = 81

et : −34 = −3 × 3 × 3 × 3 = −81

🧮 Opérations sur les puissances

Avec n et p entiers relatifs :

  • an × ap = a(n+p)
  • an / ap = a(n−p)
  • (an)p = a(n×p)
  • (a × b)n = an × bn
  • a−1 = 1 / a
  • a−n = 1 / an

🛠️ Méthode — Calculs sur les puissances

Exprimer sous la forme d’une seule puissance

A = 1 / 42     B = 45 × 47     C = 54 / 56     D = 73 × (72)6     E = 67 × 97

✅ Correction

A = 1 / 42 = 4−2
B = 45 × 47 = 45+7 = 412
C = 54 / 56 = 54−6 = 5−2
D = 73 × (72)6 = 73 × 712 = 715
E = 67 × 97 = (6×9)7 = 547

🔟 Puissances de 10

a) Écrire sous la forme 10n ou 10−n

A = 104 × 107
B = 10−4 / 105
C = (102)−6
D = 10−4 × (103)−1

✅ Correction a)

A = 104+7 = 1011
B = 10−4−5 = 10−9
C = 102×(−6) = 10−12
D = 10−4 × 103×(−1) = 10−4−3 = 10−7

b) Écrire en notation scientifique

A = 4 × 7 × 10−5 × 10−8
B = (7 × 10−4 × 5 × 108) / (56 × 10−9)
C = (32 × 10−4 + 6 × 10−3) / (2 × 10−5)

✅ Correction b)

A = 28 × 10−13 = 2,8 × 10−12

B = (7×5)/56 × (10−4×108) / 10−9 = 0,625 × 1013 = 6,25 × 1012

C = (0,0032 + 0,006) / (2 × 10−5) = 0,0092 / (2 × 10−5) = 0,0046 × 105 = 4,6 × 102
