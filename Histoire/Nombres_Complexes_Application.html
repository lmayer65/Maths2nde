TP — Une équation du troisième degré
et les nombres complexes

Objectif : comprendre pourquoi les nombres complexes peuvent apparaître naturellement dans la résolution d’une équation du troisième degré, même lorsque la solution finale est réelle.

1. L’équation étudiée

On considère l’équation :

\[ x^3 = 15x + 4 \]

c’est-à-dire :

\[ x^3 - 15x - 4 = 0 \]
Question 1. Vérifier que \(x=4\) est solution de l’équation.

2. Changement de variable

On pose :

\[ x = u+v \]

Cette écriture n’est pas unique : pour un même nombre \(x\), il existe une infinité de couples \((u;v)\) tels que \(x=u+v\).

Idée importante : comme on a une certaine liberté dans le choix de \(u\) et \(v\), on pourra imposer une condition supplémentaire pour simplifier les calculs.
Question 2. Développer \((u+v)^3\).

On utilise l’identité :

\[ (u+v)^3 = u^3 + v^3 + 3uv(u+v) \]

Comme \(x=u+v\), on obtient :

\[ x^3 = u^3 + v^3 + 3uv(u+v) \]

3. Remplacement dans l’équation

On remplace dans l’équation :

\[ x^3 = 15x+4 \]

Comme \(x=u+v\), cela donne :

\[ u^3+v^3+3uv(u+v)=15(u+v)+4 \]

On regroupe les termes :

\[ u^3+v^3+(3uv-15)(u+v)-4=0 \]

4. Choix stratégique

On veut simplifier l’expression. On choisit donc d’annuler le coefficient de \((u+v)\) :

\[ 3uv-15=0 \]
Question 3. En déduire la valeur de \(uv\).

On obtient :

\[ uv=5 \]

Avec ce choix, l’équation devient simplement :

\[ u^3+v^3=4 \]
On cherche donc maintenant deux nombres \(u\) et \(v\) tels que : \[ \begin{cases} u^3+v^3=4\\ uv=5 \end{cases} \]

5. Apparition de \(\sqrt{-1}\)

On introduit le nombre \(i\), défini par :

\[ i^2=-1 \]

On propose alors le couple :

\[ u=2+i \qquad \text{et} \qquad v=2-i \]
Question 4. Calculer \(uv=(2+i)(2-i)\).

On obtient :

\[ uv=(2+i)(2-i)=4-i^2=4+1=5 \]

La première condition est donc bien vérifiée :

\[ uv=5 \]
Question 5. Calculer \((2+i)^3\).

On trouve :

\[ (2+i)^3=2+11i \]
Question 6. Calculer \((2-i)^3\).

On trouve :

\[ (2-i)^3=2-11i \]

Donc :

\[ u^3+v^3=(2+11i)+(2-11i)=4 \]

La deuxième condition est aussi vérifiée :

\[ u^3+v^3=4 \]

6. Retour à \(x\)

Comme \(x=u+v\), on obtient :

\[ x=(2+i)+(2-i) \]

Les parties imaginaires disparaissent :

\[ x=4 \]
Conclusion : les nombres complexes sont apparus dans les calculs intermédiaires, mais la solution finale est un nombre réel.

7. Ce qu’il faut retenir

Dans les réels, la fonction cube est strictement croissante. Ainsi, une équation du type :

\[ u^3=a \]

a une seule solution réelle.

Mais dans les complexes, une équation du type :

\[ u^3=z \]

a en général plusieurs solutions complexes.

Cela montre que les règles connues dans les réels ne se généralisent pas toujours directement aux nombres complexes.

Corrigé rapide

Question 1

\[ 4^3-15\times 4-4=64-60-4=0 \]

Donc \(x=4\) est solution.

Question 2

\[ (u+v)^3=u^3+3u^2v+3uv^2+v^3 \]
\[ (u+v)^3=u^3+v^3+3uv(u+v) \]

Question 3

\[ 3uv-15=0 \]
\[ 3uv=15 \]
\[ uv=5 \]

Question 4

\[ (2+i)(2-i)=4-2i+2i-i^2=4-i^2=5 \]

Question 5

\[ (2+i)^2=4+4i+i^2=3+4i \]
\[ (2+i)^3=(3+4i)(2+i)=6+3i+8i+4i^2 \]
\[ (2+i)^3=6+11i-4=2+11i \]

Question 6

\[ (2-i)^2=4-4i+i^2=3-4i \]
\[ (2-i)^3=(3-4i)(2-i)=6-3i-8i+4i^2 \]
\[ (2-i)^3=6-11i-4=2-11i \]

Conclusion

\[ u^3+v^3=(2+11i)+(2-11i)=4 \]
\[ uv=5 \]
\[ x=u+v=(2+i)+(2-i)=4 \]

Les nombres complexes interviennent dans les calculs intermédiaires, mais la solution finale est réelle.
